Задание 1

Создать базу данных для хранения информации о студентах университета. База данных должна содержать три таблицы:
  • "Студенты"
  • "Факультеты"
  • "Оценки"
В таблице "Студенты" должны быть следующие поля:
  • id
  • имя
  • фамилия
  • возраст
  • пол
  • email
  • группа
  • id факультета
В таблице "Факультеты" должны быть следующие поля:
  • id
  • название факультета
В таблице "Оценки" должны быть следующие поля
  • id
  • id студента
  • название предмета
  • оценка
  • id факультета
Необходимо связать таблицу "Студенты" с таблицей "Факультеты" и таблицу "Студенты" с таблицей "Оценки". Написать функцию-обработчик, которая будет выводить список всех студентов с указанием их факультета и оценок. Решение задания 1

Задание 2

Создайте форму регистрации пользователя с использованием Flask-WTF. Форма должна содержать следующие поля:
  • имя пользователя (обязательное поле)
  • электронная почта (обязательное поле, с валидацией на корректность ввода email)
  • дата рождения(обязательное поле, с валидацией на дату в формате дд.мм.гггг)
  • пароль (обязательное поле, с валидацией на минимальную длину пароля в 8 символов и содержание хотя бы одной буквы и одной цифры)
  • подтверждение пароля (обязательное поле, с валидацией на совпадение с паролем)
После отправки формы данные должны сохраняться в базе данных (можно использовать SQLite) и выводиться сообщение об успешной регистрации. Пароль должен быть зашифрован. Если какое-то из обязательных полей не заполнено или данные не прошли валидацию, то должно выводиться соответствующее сообщение об ошибке. Добавить проверку на уникальность имени пользователя и электронной почты в базе данных. Если такой пользователь уже зарегистрирован, то должно выводиться сообщение об ошибке. Решение задания 2

Задание 3

Создать базу данных для хранения информации о книгах в библиотеке. База данных должна содержать две таблицы:
  • "Книги"
  • "Авторы"
В таблице "Книги" должны быть следующие поля:
  • id
  • название
  • год издания
  • количество экземпляров
  • id автора
В таблице "Авторы" должны быть следующие поля:
  • id
  • имя
  • фамилия
Необходимо создать связь между таблицами "Книги" и "Авторы". Написать функцию-обработчик, которая будет выводить список всех книг с указанием их авторов. У книги может быть несколько авторов, также как и у автора может быть несколько книг. Решение задания 3
